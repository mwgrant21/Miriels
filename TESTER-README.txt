TAROT & ORACLE — Tester Setup
==============================

Requirements
------------
- macOS 12 or later
- Node.js (https://nodejs.org) — download the LTS version

First-time setup
----------------
1. Unzip this folder somewhere convenient (Desktop is fine)
2. Right-click "start-mac.command" -> Open
   (The first time you may see a security warning — click Open anyway)
3. A terminal window will open, install dependencies, and launch the app

After that
----------
Double-click "start-mac.command" to launch anytime.
The terminal window can be minimized — keep it open while using the app.

Anthropic API Key
-----------------
The app uses Claude AI for card interpretations.
Click the gear icon (⚙) in the top-right corner to enter your API key.
Keys are stored locally on your machine only.

Questions?
----------
Close the terminal window to quit the app.
